# Zippo

macOS Dock app which bridges [google/fuse-archive](https://github.com/google/fuse-archive) and **macOS**.

## Features

- [ ] drop archives (zip, tar.gz, or otherwise) on Dock icon and mount them according to _rules_ and _workflows_
- [ ] show what mounted and where with metadata
- [ ] support unecrytped archives
- version 0.2
- [ ] support ecrytped archives
